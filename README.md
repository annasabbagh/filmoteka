# :clapper: <a href="https://annasabbagh.github.io/filmoteka/" target="_blank"><img src="" alt=""/> Filmoteka</a>

* Adaptive interface (mobile, tablet, desktop).
* Project was developped in 13 days by 7 collaborators.
